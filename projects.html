Featured software project filepackR File. Pack. Route.

Cleaner document workflows with less repetitive work.

filepackR is PDF splitter, merger, and document-processing software that turns mixed files into cleaner, audit-ready records. It uses OCR-powered recognition, customizable document rules, review queues, and Intelligent File Routing to move documents through the right workflows.

  • ✓Split scanned PDF packets into separate documents
  • ✓Merge related pages and supporting records
  • ✓Apply consistent filenames and routing rules
  • ✓Flag review items instead of silently guessing
  • ✓Support local-first processing for sensitive files
